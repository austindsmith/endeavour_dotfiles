# Endeavour OS dotfiles

Trying Endeavour out on my laptop, and figured I'd rebuild my dotfiles cleanly with lessons learned and not break any configurations on my main computer until it's ready to transfer.

## Documentation and configuration guides

[GNU Stow](https://www.gnu.org/software/stow/manual/stow.html) - Easier management of dotfiles
[Hyprland](https://wiki.hypr.land/) Fun new compositor
[Kitty](https://sw.kovidgoyal.net/kitty/conf/) Terminal emulator, simple, fast
[Starship](https://starship.rs/guide/) Terminal prompt customization
[Navi](https://navi.readthedocs.io/en/latest/getting_started.html) Terminal cheatsheets
[Fastfetch](https://github.com/fastfetch-cli/fastfetch/wiki/Configuration) System information
[Paru](https://github.com/Morganamilo/paru) Package manager, improved version of `yay`
[Atun](https://docs.atuin.sh/cli/) Shell history
[Bat](https://github.com/sharkdp/bat?tab=readme-ov-file#configuration-file) Cat with more features
[Cliphist](https://github.com/sentriz/cliphist) Clipboard history
[Micro](https://github.com/zyedidia/micro/blob/master/runtime/help/options.md) Terminal text editor with intuitive commands
[Btop](https://github.com/aristocratos/btop) - System resources
[Rofi applets](https://github.com/adi1090x/rofi)
[Awesome rofi](https://github.com/cjbassi/awesome-rofi)
[Pywal](https://itsfoss.com/pywal/) Generate colors from images
